#### Computing Resources Statement
Dr. Yuan’s team is the primary user group for the institutional Kubernetes (k8s) cluster, which offers rich GPU resources, including 136 NVIDIA A100s and 48 NVIDIA H100s, alongside 682 AMD EPYC CPU cores. The team has 380 TB of institutional storage with on-demand expansion. In addition, Dr. Yuan's team operates six dedicated Lambda GPU workstations (“T6” nodes) exclusively for team personnel’s usage. Each T6 features two NVIDIA RTX 4090 (24 GB) GPUs, a 32-core AMD Ryzen Threadripper PRO 5955WX, 256 GB RAM, and 11 TB of local SSD storage.
